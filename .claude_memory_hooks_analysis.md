# Hooks Refactoring Analysis - useKeyboardShortcuts

## Current Structure (449 lines)
The useKeyboardShortcuts hook contains multiple concerns that can be separated:

### Distinct Concerns Identified:
1. **Event Handling** (lines 96-127, 237-262)
   - Shortcut parsing (`parseShortcut`)
   - Event matching (`matchesShortcut`) 
   - Keyboard event listeners setup

2. **Focus Management** (lines 85-91, 171-185)
   - Input element detection (`isInputElement`)
   - Context validation (`canProcessShortcuts`)
   - Respect for input fields

3. **Shortcut Registry** (lines 69-74, 133-166)
   - Registration/unregistration (`registerShortcut`, `unregisterShortcut`)
   - Active shortcuts tracking
   - Conflict resolution and state management

4. **Execution Engine** (lines 190-232)
   - Action execution (`executeShortcut`)
   - Error handling
   - Screen reader announcements

### Dependencies Analysis:
- **External**: `useRouter` (Next.js), `useScreenReaderAnnouncements` (custom)
- **DOM**: Document/window event listeners, activeElement detection
- **State**: React useState for shortcuts Map and activeShortcuts Set

### Provider Integration:
- No direct app context dependencies
- Uses screen reader announcement hook (accessibility)
- Router integration for navigation shortcuts

### Refactoring Plan:
1. Extract `KeyboardEventService` - pure functions for event handling
2. Extract `FocusManagerService` - focus/context management 
3. Extract `ShortcutRegistryService` - registry with conflict resolution
4. Compose simplified hook using these services
5. Maintain exact same public API for backward compatibility

### Backward Compatibility Requirements:
- Preserve all public methods: `registerShortcut`, `unregisterShortcut`, `isShortcutActive`, `getShortcuts`, `activeShortcuts`
- Maintain same options interface: `UseKeyboardShortcutsOptions`
- Keep same behavior for `useGlobalKeyboardShortcuts` and `useContextualKeyboardShortcuts`

### Service Architecture:
Following existing `/lib/services/` pattern with transformer/validator services
